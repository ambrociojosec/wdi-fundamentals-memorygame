Instructions

Concentration, also know as Match Match, Memory, Pelmanism, Shinkeisuijaku, Pexeso or simply Pairs, is a card game in which all of the crads are laid face down on a surface and two cards are flipped face up ove each turn. The object of the game is to turn over pairs of matching cards.
